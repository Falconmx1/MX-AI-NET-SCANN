## 🚀 Características

- ✅ **Escaneo de hosts** - Detección de dispositivos activos mediante ping
- 🔌 **Detección de puertos** - Escaneo de 21 puertos comunes
- 🧠 **Análisis IA** - Evaluación de riesgos y recomendaciones inteligentes
- 🌐 **Interfaz web** - Resultados en tiempo real con diseño cyberpunk
- 🎯 **Sin dependencias pesadas** - Solo Flask y requests

## 📋 Requisitos

- Python 3.8 o superior
- Conexión de red (para escanear)

## 🔧 Instalación

```bash
# Clonar repositorio
git clone https://github.com/Falconmx1/MX-AI-NET-SCANN.git
cd MX-AI-NET-SCANN

# Instalar dependencias

💻 Uso
# Iniciar servidor
python app.py
